# 球探比分——体育组队预约

  "球探比分——体育组队预约"是一款线上预定球场(网球/篮球)和发布组队、加入赛事的应用。
  
  1.在线预约球场(网球/篮球)以及发布组队、加入赛事等;
  2.首页精选网球场和篮球场下的热门球场;
  3.球场预约:添加预约时间、预约时间、人数、以及联系人信息可提交预约信息;
  4.球场的计费方式以人数计算，与预约小时数无关，且最长4个小时;
  5.预约成功将生成订单编号,可根据订单编号到店消费;
  6.寻找组队:寻找该球场下的赛事列表,可选择性的选择赛事进行组队;
  7.发起赛事:用户可发起赛事,让其他用户加入组队，进行约球;
  8."我的":修改头像/昵称,查看我发起的赛事,我参与的赛事,我的收藏,密码修改和退出登录等.
